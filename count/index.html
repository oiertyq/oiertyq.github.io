A.count

题意

给定两个整数 $n$,$m$, 求 k 元组 $(a_1,a_2,…,a_k)$ 的个数，满足$a_1,a_2,…,a_k$ 为正整数,$\sum_{i=1}^{k} a_i=n$且$a_1,a_2,…,a_k$均不是$m$的倍数。

解法

设$a_i = m*b_i+c_i (c_i\in(0,m))$

设$csum = \sum c_i, \;bsum=\sum b_i$

有$csum = n \;mod \;m+p\times m\;,\; bsum=\frac{n-csum}{m};(p\in[0,k])$

枚举p， b的方案数即为$\Pi_{j=1}^{k-1}\;\;\frac{bsum+j}{j}$（这里不需组合数，暴力计算即可，预处理范围太大）

对于c的方案，考虑容斥（因为$c_i \leq m$）

容斥只需令$j$个$m \leq c_i$, 但是$c_i>1$
